# N4 French Community

The bilingual one-page website for the French and French-speaking community around North London N4.

## Local development

```sh
pnpm install
pnpm dev
```

The English page is available at `/` and the French page at `/fr/`.

## Production build

```sh
pnpm build
```

The static production site is generated in `dist/` and can be deployed to any static host. The canonical production domain is `https://n4french.co.uk`.
